## Scenarios

### Product Catalog System (HTTP/Messages)

Our standard example is a product catalog website (see [React](/docs/examples/js/consumer)) which consumes a basic Products API implementing the following HTTP interface.

All demo languages implement this interface so that we can easily mix and match technologies.

See and interact with the OpenAPI Product Spec [here](/oas/products)
